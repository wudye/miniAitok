Method 是 java.lang.reflect.Method 的实例，表示被拦截的那个方法的反射对象。通过它可以读取方法名、声明类、参数类型、返回类型和注解等。下面给出一个简短示例：先是一个控制器方法，然后在切面里获取反射对象并演示如何读取信息及如何定位到实现类的方法（以防代理导致拿到的是接口方法）。
示例说明：切面先从签名拿到一个 Method（反射对象），可以用 method.getName() / method.getDeclaringClass() / method.getAnnotation(...) 等方法读取信息；若应用使用接口代理且你需要实现类上的注解或实现方法，用 joinPoint.getTarget().getClass().getMethod(...) 在目标类上再解析一次
// java
package example;

import com.mwu.aitok.service.video.annotation.VideoRepeatSubmit;
import org.aspectj.lang.JoinPoint;
import org.aspectj.lang.reflect.MethodSignature;
import java.lang.reflect.Method;
import java.util.Arrays;

// 示例控制器
public class VideoController {
@VideoRepeatSubmit(key = "upload", interval = 5000)
public void uploadVideo(String id, byte[] data) {
// 上传实现
}
}

// 切面示例（片段）
public class ExampleAspect {

    public void beforeAdvice(JoinPoint joinPoint) throws Exception {
        // 从签名获取反射 Method（代表被调用的方法）
        MethodSignature signature = (MethodSignature) joinPoint.getSignature();
        Method reflected = signature.getMethod(); // 这就是 "Method" 对象

        // 读取一些信息
        String name = reflected.getName(); // "uploadVideo"
        Class<?> declaredOn = reflected.getDeclaringClass(); // 可能是接口或代理类
        Class<?>[] paramTypes = reflected.getParameterTypes(); // [String.class, byte[].class]

        // 读取注解（如果注解放在接口方法上，这里能拿到）
        VideoRepeatSubmit ann = reflected.getAnnotation(VideoRepeatSubmit.class);

        // 如果项目使用了接口代理且你需要实现类的 Method 或在实现类上查注解：
        Object target = joinPoint.getTarget();
        Method implMethod = target.getClass().getMethod(reflected.getName(), paramTypes);
        Class<?> implDeclaring = implMethod.getDeclaringClass(); // 真正的实现类
        VideoRepeatSubmit implAnn = implMethod.getAnnotation(VideoRepeatSubmit.class); // 实现类上的注解（若有）

        // 示例打印（伪输出）
        System.out.println("reflected.getName(): " + name);
        System.out.println("reflected.getDeclaringClass(): " + declaredOn.getName());
        System.out.println("paramTypes: " + Arrays.toString(paramTypes));
        System.out.println("implDeclaringClass: " + implDeclaring.getName());
        System.out.println("annotation from reflected: " + ann);
        System.out.println("annotation from impl: " + implAnn);
    }
}

/*
可能的输出示例：
reflected.getName(): uploadVideo
reflected.getDeclaringClass(): com.example.VideoApi (若通过接口)
implDeclaringClass: com.example.VideoControllerImpl
annotation from reflected: null
annotation from impl: @VideoRepeatSubmit(...)
*/
